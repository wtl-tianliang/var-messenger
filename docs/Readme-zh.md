## 功能特性

- 批量发送

支持使用`.xlsx`文件定义变量，设置多个发送地址，向多个邮箱投递邮件。
![post](./static/post.png)

- 变量正文

支持在邮件正文中使用变量，发送邮件时自动替换为正确的内容。
![define-var](./static/define-var.png)

- 模板附件

支持在使用`.docx`文件作为附件时，自动填充替换附件内的变量。
![word-template](./static/word.png)
